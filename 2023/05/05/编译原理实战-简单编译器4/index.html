基础信息

上一个版本已经实现了结构体，此版本主要实现指针

语法分析

  • 把指针 IDP 当成 ID 处理即可
  • 在 Exp 中添加对应的规则（用于识别 &a 和 *a）
  • PS：这里使用的是 ID，这就意味着结构体中的指针没法被识别（暂时不做修改）

语义分析

指针声明（全局）

  • 和 ID 类似的处理方式

指针声明（局部）

  • 和 ID 一起处理，但遇到结构体指针时需要特殊处理
